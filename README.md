# Local Development

The following assumes the use of `node@>=10`.

Current version on (https://www.npmjs.com/package/@sd_uniswap/v2-core).

## Install Dependencies

`yarn`

## Compile Contracts

`yarn compile`

## Run Tests

`yarn test`
